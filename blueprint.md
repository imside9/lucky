# 오늘의 운세 웹사이트

## 1. 개요

오늘의 날짜를 기반으로 사용자가 선택한 종류(예: 띠별, 별자리)에 따라 운세를 보여주는 웹 애플리케이션입니다. 운세 정보는 오픈 API 또는 AI를 통해 동적으로 생성됩니다.

## 2. 주요 기능

*   **오늘 날짜 표시:** 화면 상단에 항상 오늘 날짜가 표시됩니다.
*   **운세 종류 선택:** 사용자는 띠별 운세, 별자리 운세 등 원하는 운세 종류를 선택할 수 있습니다.
*   **운세 결과 표시:** 선택한 종류에 맞는 오늘의 운세 결과를 보여줍니다.
*   **모바일 반응형 디자인:** 다양한 화면 크기에서 최적화된 UI를 제공합니다.

## 3. 기술 스택 및 디자인

*   **Front-end:** HTML, CSS, JavaScript (프레임워크 없이 순수 기술 사용)
*   **핵심 기술:** 웹 컴포넌트(Web Components)를 사용하여 UI를 모듈화하고 재사용성을 높입니다.
*   **데이터:** 외부 오픈 API 또는 AI 기반 생성 API를 연동하여 운세 정보를 가져옵니다.
*   **디자인:**
    *   **색상:** 신비롭고 편안한 느낌을 주는 어두운 계열의 색상(예: 남색, 보라색)을 메인으로 사용하고, 포인트 색상으로 금색 또는 은색을 활용합니다.
    *   **폰트:** 가독성이 좋으면서도 신비로운 느낌을 주는 세리프(Serif) 계열의 폰트를 사용합니다.
    *   **레이아웃:** 카드 형태의 UI를 사용하여 각 운세 정보를 명확하게 구분하고, 아이콘과 적절한 여백을 활용하여 사용자가 정보를 쉽게 인지할 수 있도록 디자인합니다.
    *   **애니메이션:** 부드러운 전환 효과(Fade-in/out)와 미묘한 애니메이션을 추가하여 사용자 경험을 향상시킵니다.

## 4. 현재 진행 계획

1.  **`blueprint.md` 파일 생성:** 프로젝트 설계 및 계획 문서화 (완료)
2.  **기본 HTML 구조 설계:** 날짜, 운세 종류 선택, 결과 표시를 위한 기본 UI 구성
3.  **기본 CSS 스타일링:** 초기 디자인 적용
4.  **기본 JavaScript 로직 구현:** 날짜 표시 및 사용자 인터랙션 처리
5.  **운세 API 탐색 및 연동:** 운세 정보를 가져올 API 탐색 및 연동
6.  **웹 컴포넌트 리팩토링:** UI를 웹 컴포넌트로 분리하여 모듈화
