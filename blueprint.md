# 영단어 퀴즈 & 고양이 숏츠 (Word Quiz & Cat Shorts)

## 1. 개요

사용자가 중학생 수준의 영단어 3개를 맞히면, 보상으로 귀여운 고양이 유튜브 숏츠 영상 1개를 볼 수 있는 웹 애플리케이션입니다. 학습과 재미를 결합하여 동기를 부여합니다.

## 2. 주요 기능

*   **영단어 퀴즈 (3단계):**
    *   총 3개의 영단어 퀴즈가 순서대로 제시됩니다.
    *   난이도는 대한민국 중학생 필수 영단어 수준으로 설정합니다.
    *   영어 단어를 보여주고 한국어 뜻을 입력하거나, 한국어 뜻을 보여주고 영단어를 입력하는 방식입니다. (힌트 없이 글자 수만 표시)
*   **보상 시스템 (고양이 숏츠):**
    *   3문제를 모두 정답 처리하면 보상 화면으로 전환됩니다.
    *   유튜브에서 인기 있는 귀여운 고양이 숏츠(Shorts) 영상 1개를 볼 수 있습니다.
*   **반응형 UI:** PC와 모바일 환경 모두에서 쾌적하게 퀴즈를 풀고 영상을 시청할 수 있습니다.

## 3. 기술 스택 및 디자인

*   **Front-end:** HTML, CSS, JavaScript (프레임워크 없이 순수 기술 사용)
*   **API:**
    *   **Word Data:** 영단어 제공 API (예: Datamuse API, Random Word API)를 사용하거나, 난이도 조절을 위해 AI를 통해 미리 생성된 중학생 수준 단어셋을 활용합니다.
    *   **YouTube Data API:** 'cute cat shorts', 'funny cat' 등의 키워드로 인기 영상을 검색하여 가져옵니다.
*   **디자인:**
    *   **레퍼런스:** [Insurance Comparison Mobile iOS App Design](https://dribbble.com/shots/27083220-Insurance-Comparison-Mobile-iOS-App-Design)
    *   **컨셉:** 레퍼런스의 깔끔하고 모던한 카드형 UI를 기반으로, 고양이 테마에 맞는 귀여운 포인트 추가.
    *   **레이아웃:** 둥근 모서리(Rounded Corners)가 적용된 카드 형태의 컨테이너 사용.
    *   **색상:** 레퍼런스의 화이트/라이트 그레이 배경에 파스텔 톤(핑크, 옐로우)을 포인트 컬러로 사용하여 부드러운 분위기 연출.
    *   **폰트:** 가독성이 좋은 산세리프(Sans-serif) 폰트 (예: Pretendard, Jua 등).
