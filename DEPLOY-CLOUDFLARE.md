# Cloudflare Stage Deploy Guide

## 1) One-time setup

1. Install Node.js 18+.
2. Login to Cloudflare.
```bash
npm run cf:login
```
3. In Cloudflare Dashboard, create a Pages project once.
- Example project name: `tap-challenge-demo`

## 2) Local preview

```bash
npm run preview
```

Open: `http://localhost:8787`

## 3) Stage deploy

```bash
CF_PAGES_PROJECT=tap-challenge-demo CF_BRANCH=stage npm run cf:deploy:stage
```

If `CF_PAGES_PROJECT` is omitted, default is `tap-challenge-demo`.

## 4) Production deploy

```bash
CF_PAGES_PROJECT=tap-challenge-demo npm run cf:deploy:prod
```

## 5) Notes

1. This project is static, so `index.html`, `style.css`, and `main.js` are deployed as-is.
2. Stage URL is generated by Cloudflare Pages for the `stage` branch deployment.
3. Clipboard sharing may fail on non-HTTPS local preview; it works best on deployed HTTPS pages.
